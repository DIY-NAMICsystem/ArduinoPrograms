
# Paradigm: Continuous Cue

___

### Simple Description

- Pseudo-random activation of Left/Middle/Right port
- LED Duration: 2 min (120,000ms)
- Inter-trial Interval (ITI): 50ms
- Cue stays on UNTIL mouse makes a response (nosepoke)
